SugarCRM Plugin for OpenMeetings

You can get the latest Version of the Plugin for SugarCRM from the downloads .

You can find the OpenMeetings Plugin also in the SugarCRM Plugin database .

A demo version of the SugarCRM plugin is available at demo.e-schwert.de . Username: demo Password: Demouser55


Features

The plugin contains the following features:

  • Create videoconference rooms through SugarCRM:
    1. Integrated with calendar
    2. Integrated with SugarCRM meetings
    3. Sends invetations emails to meeting participants
    4. Configurable room type


Installation the SugarCRM Plugin for OpenMeetings

In SugarCRM's administration web-interface you will find the module loader.

  1. Upload the module with module loader in administration area.
  2. Configure OpenMeetings settings.
  3. Create OpenMeetings rooms.
