1. Recomendações para o ambiente de produção

    Por padrão o banco de dados Apache Derby integrado ao sistema é utilizado. Para o ambiente de produção você deve considerar o uso do MySQL, Postgres ou, por exemplo, IBM DB2 ou Oracle

  2. Habilitando envio e importação de imagens para a lousa

    • Instale ImageMagick no servidor, você pode obter maiores informações em http://www.imagemagick.org, inclusive sobre a instalação. As instruções sobre a instalação podem ser encontradas aqui http://www.imagemagick.org/script/binary-releases.php, entretanto na maioria dos sistemas Linux você poderá instalá-lo pelo seu gerenciador de pacotes favorito (ex.: apt-get)

  3. Habilitando importação de PDFs na lousa

    • Instale GhostScript no servidor, você pode obter maiores informações em http://pages.cs.wisc.edu/~ghost/, inclusive sobre a instalação. As instruçõs sobre a instalação podem ser encontradas neste mesmo link, entretanto na maioria dos sistemas Linux você poderá instalá-lo pelo seu gerenciador de pacotes favorito (ex.: apt-get)
    • Instale SWFTools no servidor, você pode obter maiores informações em http://www.swftools.org/, inclusive sobre a instalação. Algumas distribuições de Linux já o possui em seus gerenciadores de pacote, veja http://packages.debian.org/unstable/utils/swftools. A versão recomendada do SWFTools é 0.9, pois a versão anterior possui um bug que coloca o objeto na lousa com dimensões incorretas

  4. Habilitando importação de todos os documentos Office como .doc, .docx, .ppt, .pptx, (...) na lousa

    • OpenOffice-Service iniciado e executando na porta 8100, veja OpenOfficeConverter para detalhes

  5. Habilitando Gravações e importação de arquivos .avi, .flv, .mov e .mp4 na lousa

    • Instale FFMpeg. Você deve obter a versão mais atual do FFMPEG! Para Windows você pode baixar a vers√£o compilada, por exemplo, de http://ffmpeg.arrozcru.org/builds/ Usuários Linux ou OSx deverão usar uma das diversas instruções de instalação existentes na Web. Você precisa habilitar o libmp3lame!
    • Instale SoX http://sox.sourceforge.net/. Você deve instalar a versão mais recente do SoX! SoX 12.xx NÃO funcionará!
